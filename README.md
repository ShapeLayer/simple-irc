# Simple IRC

![](./.static/demo.png)

The implementation of task's requirements for 〈Network Systems(ネットワークシステム)〉 course at Saga University.

## Getting Started

```sh
cd client
mkdir build
cmake ..
make
```

```sh
cd server
mkdir build
cmake ..
make
```

```
chmod u+x ./container
./container build
./container run
```
